# Seminar Pointer

オンラインセミナーや画面共有中に、画面の上から「ここを見て」「ここが重要」をすばやく示すための macOS 向け注釈ツールです。

Zoom、Google Meet、Chrome、ChatGPT、Codex、コードエディターなど、普段使っているアプリの上に半透明のオーバーレイを出して、枠・矢印・テキスト・注目マーカーを重ねられます。

## 対応環境

- macOS 専用です
- 開発・確認環境: macOS + Electron
- Windows 版は現時点では未対応です

## 主な機能

- 四角、丸、線、矢印の描画
- ①②③、日本語、任意テキストのスタンプ配置
- スモーキーピンクの「ここ見て！」注目マーカー
- Control キー 2 回で、カーソル位置に「ここ見て！」を表示
- ⌘Z で 1 つ前の注釈を取り消し
- ⌘Shift+C で全消去
- ツールバーとテキストパネルのドラッグ移動
- 起動中、閉じる、終了が分かるツールバー表示
- Finder から起動できる簡易 `.app` ランチャー

## 使い方

1. `Seminar Pointer.app` をダブルクリックします
2. 画面に何も出ない状態で起動します
3. **⌘ + Shift + S** を押すと、左下にツールバーが表示されます
4. ツールを選んで、画面上に注釈を描きます
5. もう一度 **⌘ + Shift + S** を押すと、注釈モードを閉じます
6. 完全に終了したいときは、ツールバーの **終了** を押します

起動中でも、オーバーレイを閉じている間は背後のアプリを普通に操作できます。

## ショートカット

| キー | 動作 |
| --- | --- |
| ⌘ + Shift + S | オーバーレイの表示 / 非表示 |
| ⌘ + Shift + C | 全消去 |
| ⌘ + Z | 1 つ前に戻る |
| Control 2 回 | カーソル位置に「ここ見て！」 |
| Esc | 操作なしモードに戻る |

## ツールバー

| ボタン | 動作 |
| --- | --- |
| 操作なし | 何も描かない状態に戻ります |
| 四角 | 赤枠などで囲みます |
| 丸 | 円・楕円で囲みます |
| 線 | 線を引きます |
| 矢印 | 矢印を引きます |
| テキスト | ①②③や日本語テキストを置きます |
| ここ見て | クリック位置に注目マーカーを出します |
| 全消去 | 画面上の注釈をすべて消します |
| 閉じる | オーバーレイを閉じます |
| 終了 | アプリを完全終了します |

ツールバー左端の `⋮⋮` をドラッグすると、好きな位置へ移動できます。テキストパネルも同じように移動できます。

## 配布について

このリポジトリは、現在は開発用の簡易配布形式です。

コミュニティの人に渡すだけなら、次のどちらかで使えます。

- このリポジトリをダウンロードしてもらい、`npm install` 後に `Seminar Pointer.app` を起動
- ZIP でフォルダごと渡して、同じく `npm install` 後に起動

ただし、まだ正式なインストーラーや署名済みアプリではありません。Mac に慣れていない人へ配る場合は、今後 `electron-builder` などで正式な `.app` / `.dmg` にするのがおすすめです。

## 開発者向けセットアップ

```bash
npm install
```

開発起動:

```bash
npm run dev
```

ビルド確認:

```bash
npm run build
```

## Finder から起動できない場合

`Seminar Pointer.app` で起動できない場合は、同じフォルダにある `Seminar Pointer.command` をダブルクリックしてください。

それでも起動しない場合は、ターミナルで以下を実行してください。

```bash
npm install
npm run dev
```

起動ログは `seminar-pointer-launch.log` に出力されます。

## macOS の権限

初回起動時や画面共有時にうまく表示されない場合は、以下を確認してください。

- システム設定 > プライバシーとセキュリティ > アクセシビリティ
- システム設定 > プライバシーとセキュリティ > 画面収録

`Electron`、`Terminal`、または起動に使ったアプリを許可する必要がある場合があります。

## Zoom / Meet で使うとき

Zoom では、画面共有時に **Desktop / デスクトップ全体** を共有すると注釈が共有先にも見えやすいです。特定のアプリウィンドウだけを共有すると、オーバーレイが相手に映らないことがあります。
