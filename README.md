# Seminar Pointer

オンラインセミナーの画面共有中に、画面上へ注釈・図形・テキスト・注目マーカーを重ねて表示できるデスクトップアプリです。

## 主な機能

- 四角・丸・線・矢印の描画
- ①②③などのテキストスタンプ配置
- 「ここ見て！」注目マーカー（波紋アニメーション）
- 全消去
- ショートカットキー操作

## セットアップ

```bash
npm install
```

## 起動方法

Finder から簡単に起動する場合:

1. `Seminar Pointer.app` をダブルクリック
2. 起動後、**⌘ + Shift + S** でオーバーレイを表示

うまく開けない場合は `Seminar Pointer.command` をダブルクリックしてください。

ターミナルから起動する場合:

```bash
npm run dev
```

または:

```bash
npm start
```

※ 初回だけ macOS に確認される場合があります。その場合は「システム設定 > プライバシーとセキュリティ」で許可してください。

## 操作方法

1. アプリを起動するとバックグラウンドで待機状態になります（画面は完全に透過）
2. **⌘ + Shift + S** でオーバーレイを表示（注釈モードON）
3. 右側に表示されるツールバーからツールを選択
4. 画面上でドラッグして図形を描画
5. テキストツール選択後、パネルで文字を選んでクリックで配置
6. 「ここ見て」ツール選択後、クリックした位置に波紋アニメーション表示
7. 全消去ボタンまたは **⌘ + Shift + C** で注釈を全削除
8. **⌘ + Shift + S** または「閉じる」ボタンでオーバーレイを非表示

## ショートカット

| キー | 動作 |
|------|------|
| ⌘ + Shift + S | オーバーレイ 表示 / 非表示 |
| ⌘ + Shift + C | 全消去 |
| Esc | 現在のモード解除（四角ツールに戻る） |
